# LMS Management System

#### A comprehensive Learning Management System (LMS) built with Flask that helps educational institutes manage students, courses, staff, and learning activities efficiently.

## Features
 #### Student Registration & Management
#### Secure Login & Authentication
#### Course Management
#### Staff & Coordinator Accounts
#### Admin Dashboard
#### User Role Management
#### Course Enrollment
#### Learning Resources Upload
#### Responsive User Interface
#### Database Integration
#### Easy Deployment
## Technologies Used
#### Python
#### Flask
#### HTML5
#### CSS3
#### Bootstrap
#### JavaScript
#### SQLite
